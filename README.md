## 🔧 KUSTOMISASI

### Ganti Warna
Edit variabel CSS di `css/ultra.css`:
```css
:root {
    --neon-cyan: #00ffcc;
    --neon-magenta: #ff00aa;
    --neon-purple: #7b2dff;
    ...
}
```

### Ganti Logo
Edit SVG di HTML atau ganti dengan logo kustom.

### Tambah Bahasa
Edit `js/i18n.js` dan tambahkan translations baru.
